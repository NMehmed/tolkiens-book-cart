# Tolkiens Book Cart

This is example solution for [this kata](https://www.codewars.com/kata/59a2666349ae65ea69000051/train/ruby)

## Wh–µre to start

- First you need bundler in order to install dependencies 

```console
gem install bundler
```

- Install those

```console
bundle install
```

- Than you can run spec

```console
rspec
```
